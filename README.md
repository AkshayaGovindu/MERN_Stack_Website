# 🤖 The Robotics Club - SNIST

## Day 1 - Registration Page

I was excited to be a part of **The Robotics Club - SNIST**, which inspired me to build a web application for the club. The idea is to create a platform where students can register to join the club, and club members can securely log in to view and manage the registration details.
